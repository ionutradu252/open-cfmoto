This app tries to provide support for Android Auto on CFMoto bikes with MotoPlay support.

Everything that has been done is a hack, it should not be considered stable or production ready (and probably never will). 
**This is just a PoC!**

Having only a CFMoto 675 SR-R I cannot grant it will work on other bikes, but probably it will have problems on that same model too.

This app uses Android Auto self-mode implemented by https://github.com/andreknieriem/headunit-revived/ and relies a lot on its code.
Huge thanks and mad props to the author Andre Rinas (https://github.com/andreknieriem) for its work!

Feel free to create forks, open PR, open issue and/or bug reports.

However keep in mind that this is just a side project done in my free time.

For more informations check https://www.reddit.com/r/cfmoto/comments/1uuu63m/working_on_android_auto_on_cf_moto_at_least_675/
